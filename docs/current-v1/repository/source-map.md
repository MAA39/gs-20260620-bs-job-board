| layer | entrypoint | е®ҹйҡӣгҒ®еҪ№еүІ |
|---|---|---|
| Web Worker | TanStack Start server entry | SSRгҖҒrouteгҖҒServer Function |
| API Worker | `apps/api/src/index.ts` | Hono routerгҖҒauthгҖҒthread API |
| Agent Worker | Flue generated build | healthгҖҒFlue workflow/agent routing |
| Contracts | `packages/contracts/src/index.ts` | compile-timeе…ұжңүеһӢ |
| DB | `packages/db/src/index.ts` | query export |
| Legacy Agent package | `packages/agent/src/index.ts` | жңӘдҪҝз”ЁAI utility export |

- `apps/agent/dist/**`: Flue Cloudflare build output
- `.flue-vite/**`, `.wrangler/**`, `.output/**`: local/build artifact
